Click2Book es una aplicación web desarrollada como parte del curso Programación para la Web, diseñada para permitir a los usuarios agendar reservas en pensiones por día, semana o mes de forma rápida y sencilla.
Este repositorio corresponde al frontend del proyecto, desarrollado con Angular (standalone) por un grupo de 3 estudiantes como trabajo transversal

Funcionalidades principales: 
-Visualización de pensiones disponibles.
-Reservas por día, semana o mes.
-Interfaz moderna y clara.
-Navegación fluida entre secciones como inicio, login, pensiones, registro, etc.
